# Kurulum

Sign Platform'u çalıştırmak için kurulum rehberi.

## Sign API

```bash
git clone https://github.com/your-repo/sign-api.git
cd sign-api
mvn clean install
mvn spring-boot:run
```

## Sign UI

```bash
cd sign-ui
yarn install
yarn dev
```

Detaylar için ilgili proje README dosyalarına bakın.
