# DSS Raporları

DSS (Digital Signature Service) kütüphanesinin detaylı raporlarını almak için özel endpoint'ler kullanılabilir.

## SimpleReport (JSON)

DSS'in SimpleReport'unu JSON formatında döndürür.

### PAdES SimpleReport

```
POST /api/v1/verify/pades/report/simple
```

**Request Parameters:**

| Parametre | Tip | Zorunlu | Varsayılan | Açıklama |
|-----------|-----|---------|------------|----------|
| `signedDocument` | MultipartFile | ✅ | - | İmzalı PDF belgesi |
| `validateCertificateChain` | Boolean | ❌ | `true` | Sertifika zinciri doğrulaması |
| `checkRevocation` | Boolean | ❌ | `false` | İptal kontrolü |

**Response:** JSON formatında SimpleReport

### XAdES SimpleReport

```
POST /api/v1/verify/xades/report/simple
```

**Request Parameters:**

| Parametre | Tip | Zorunlu | Varsayılan | Açıklama |
|-----------|-----|---------|------------|----------|
| `signedDocument` | MultipartFile | ✅ | - | İmzalı XML belgesi |
| `originalDocument` | MultipartFile | ❌ | - | Orijinal belge (detached için) |
| `validateCertificateChain` | Boolean | ❌ | `true` | Sertifika zinciri doğrulaması |
| `checkRevocation` | Boolean | ❌ | `false` | İptal kontrolü |

**Response:** JSON formatında SimpleReport

## Örnek Kullanım

### SimpleReport (JSON)

```bash
curl -X POST http://localhost:8086/api/v1/verify/pades/report/simple \
  -F "signedDocument=@signed-document.pdf" \
  -F "checkRevocation=true" \
  -H "Accept: application/json"
```

### DetailedReport (XML)

```bash
curl -X POST http://localhost:8086/api/v1/verify/pades/report/detailed \
  -F "signedDocument=@signed-document.pdf" \
  -F "checkRevocation=true" \
  -H "Accept: application/xml"
```

## Rapor İçeriği

### SimpleReport

- İmza durumu (valid/invalid)
- İmza formatı
- İmzalama zamanı
- Sertifika bilgileri (temel)
- Hata ve uyarı mesajları

### DetailedReport

- Tüm SimpleReport bilgileri
- Detaylı sertifika zinciri bilgileri
- Her sertifika için doğrulama sonuçları
- İptal durumu detayları
- Zaman damgası detayları
- Validation policy sonuçları
- Qualification level bilgileri

## Response Formatları

- **SimpleReport**: JSON (application/json)
- **DetailedReport**: XML (application/xml)

## Kullanım Senaryoları

### Senaryo 1: Hızlı Doğrulama

Normal doğrulama endpoint'i kullanın (`/api/v1/verify/pades` veya `/api/v1/verify/xades`).

### Senaryo 2: Detaylı Analiz

DSS raporlarını kullanarak doğrulama sürecinin tüm detaylarını analiz edin.

### Senaryo 3: Entegrasyon

DSS raporlarını kendi sisteminize entegre edin ve özel raporlama yapın.
