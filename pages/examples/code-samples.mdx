## Python

```python
import requests

# e-Fatura imzalama
with open('efatura.xml', 'rb') as f:
    files = {'document': f}
    data = {'documentType': 'UblDocument'}
    
    response = requests.post(
        'http://localhost:8085/v1/xadessign',
        files=files,
        data=data
    )
    
    with open('signed-efatura.xml', 'wb') as out:
        out.write(response.content)
```
